# Ecossistema de Capacitação GRP (PoC)

Esta é uma Prova de Conceito (PoC) desenvolvida em Flask para um Ecossistema de Capacitação. O objetivo deste projeto é demonstrar e avaliar diferentes abordagens de navegação, busca e interação com uma base de conhecimento de Procedimentos Operacionais Padrão (POPs). 

O repositório simula dados voltados para módulos de um sistema governamental ou de gestão financeira (Empenho, Liquidação, Pagamento, Contabilidade) e seus respectivos subtemas.

## 🚀 Funcionalidades e Interfaces Demonstradas

A aplicação disponibiliza várias rotas (interfaces) distintas para apresentar e explorar os materiais de capacitação:

- **`/1_repo_linear`**: Exibição em lista plana de todos os procedimentos (Repositório Linear).
- **`/2_tree_accordion`**: Organização hierárquica por Módulo > Subtema utilizando *accordions* expansíveis.
- **`/3_tree_steps`**: Navegação guiada passo a passo (Módulo -> Subtema -> Procedimentos).
- **`/4_tree_chat`**: Interface interativa em formato de chat, com opções guiadas baseadas na árvore de conhecimento.
- **`/5_ai_semantic_chat`**: Interface de chat projetada para busca semântica conversacional (front-end de IA simulada).
- **`/6_ai_semantic_trad`**: Interface clássica de busca com barra de pesquisa utilizando aproximação semântica.
- **`/7_rag_chat`**: Interface de chat projetada para *Retrieval-Augmented Generation* (RAG) (Respostas geradas com base nos POPs recuperados).
- **`/8_rag_trad`**: Interface de busca tradicional aprimorada por IA/RAG, gerando uma resposta contextual a partir da base de POPs, em vez de exibir apenas os links.

### APIs e Integração
A aplicação também fornece dois endpoints simulados para consumo externo:
- **`POST /api/search`**: Recebe um JSON com `{ "q": "sua busca" }` e retorna os POPs mais relevantes de forma estruturada.
- **`POST /api/rag`**: Recebe uma requisição de busca e retorna uma resposta gerada contextualmente pela IA (simulada) e os POPs de embasamento.

## 🛠️ Tecnologias Utilizadas

- **Python 3.13+**
- **Flask**: Framework web para as rotas e servir as páginas.
- **Gunicorn / Werkzeug**: Servidor WSGI e dependências essenciais do Flask.
- **HTML/CSS/JS**: Interfaces de usuário modulares encontradas na pasta `templates/`.

## ⚙️ Como executar localmente

1. **Clone o repositório:**
   ```bash
   git clone <url-do-repositorio>
   cd poc-ecossistema-capacitacao-grp
   ```

2. **Crie e ative um ambiente virtual (recomendado):**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # No Linux/Mac
   # ou
   .venv\Scripts\activate  # No Windows
   ```

3. **Instale as dependências:**
   Você pode usar o instalador padrão do Python:
   ```bash
   pip install -r requirements.txt
   ```
   *Ou*, caso prefira o gerenciador rápido `uv` (pois o projeto já possui `uv.lock` e `pyproject.toml`):
   ```bash
   uv sync
   ```

4. **Execute a aplicação:**
   ```bash
   python app.py
   ```
   O servidor iniciará localmente e as rotas estarão disponíveis em: [http://127.0.0.1:5000](http://127.0.0.1:5000)

## 📁 Estrutura do Projeto

- `app.py`: Arquivo principal contendo a inicialização do Flask, geração em tempo de execução dos dados simulados (POPs), e a definição de todas as rotas web e APIs.
- `templates/`: Diretório que contém o conjunto completo de arquivos `.html` referentes às múltiplas interfaces implementadas na PoC.
- `pyproject.toml` / `uv.lock`: Configuração moderna de empacotamento Python e versionamento de dependências.
- `requirements.txt`: Lista de dependências Python congeladas (compatibilidade `pip`).
- `Procfile`: Arquivo de configuração de comandos de inicialização para deployment em plataformas PaaS (como Heroku e Render).

import os
import random
from flask import Flask, render_template, request, jsonify, session

app = Flask(__name__)
app.secret_key = "secret_key_poc_sef_mg"


# --- Dados Dummy ---
def generate_dummy_data():
    random.seed(42)
    modulos = ["Empenho", "Liquidação", "Pagamento", "Contabilidade"]
    subtemas = ["Inclusão", "Estorno", "Consulta", "Relatórios", "Configuração"]

    pops = []
    pop_id = 1
    for modulo in modulos:
        for subtema in subtemas:
            for i in range(1, 3):
                if i == 1:
                    titulo = f"{subtema} - Passo a Passo Básico"
                    texto = f"Este é o procedimento padrão para realizar a operação de {subtema.lower()} no {modulo}. Certifique-se de preencher todos os campos obrigatórios."
                else:
                    titulo = f"{subtema} - Situações de Exceção"
                    texto = f"Instruções para resolver problemas comuns e situações de exceção durante a rotina de {subtema.lower()} no {modulo}."

                link = (
                    f"youtube.com/{modulo[:3].lower()}-{subtema[:3].lower()}-{pop_id}"
                )
                pdf_name = (
                    f"POP_{modulo[:3].upper()}_{subtema[:3].upper()}_{pop_id:02d}.pdf"
                )
                pops.append(
                    {
                        "id": pop_id,
                        "modulo": modulo,
                        "subtema": subtema,
                        "titulo": f"{modulo} > {titulo}",
                        "texto": texto,
                        "link": link,
                        "pdf": pdf_name,
                    }
                )
                pop_id += 1
    return pops


POPS = generate_dummy_data()


def get_tree(pops):
    tree = {}
    for p in pops:
        m = p["modulo"]
        s = p["subtema"]
        if m not in tree:
            tree[m] = {}
        if s not in tree[m]:
            tree[m][s] = []
        tree[m][s].append(p)
    return tree


TREE = get_tree(POPS)


# --- Rotas Principais ---
@app.route("/")
def index():
    return render_template("index.html")


@app.route("/1_repo_linear")
def route_1():
    return render_template("1_repo_linear.html", pops=POPS)


@app.route("/2_tree_accordion")
def route_2():
    return render_template("2_tree_accordion.html", tree=TREE)


@app.route("/3_tree_steps")
def route_3():
    modulo = request.args.get("modulo")
    subtema = request.args.get("subtema")

    context = {
        "modulo_atual": modulo,
        "subtema_atual": subtema,
        "opcoes": [],
        "pops": [],
    }

    if not modulo:
        context["opcoes"] = list(TREE.keys())
    elif not subtema:
        if modulo in TREE:
            context["opcoes"] = list(TREE[modulo].keys())
    else:
        if modulo in TREE and subtema in TREE[modulo]:
            context["pops"] = TREE[modulo][subtema]

    return render_template("3_tree_steps.html", **context)


@app.route("/4_tree_chat")
def route_4():
    return render_template("4_tree_chat.html", tree=TREE)


@app.route("/5_ai_semantic_chat")
def route_5():
    return render_template("5_ai_semantic_chat.html")


@app.route("/6_ai_semantic_trad")
def route_6():
    query = request.args.get("q", "").lower()
    resultados = []
    if query:
        resultados = [
            p
            for p in POPS
            if query in p["titulo"].lower()
            or query in p["texto"].lower()
            or query in p["modulo"].lower()
            or query in p["subtema"].lower()
        ]
    return render_template(
        "6_ai_semantic_trad.html",
        resultados=resultados,
        query=request.args.get("q", ""),
    )


@app.route("/7_rag_chat")
def route_7():
    return render_template("7_rag_chat.html")


@app.route("/8_rag_trad")
def route_8():
    query = request.args.get("q", "").lower()
    resultados = []
    texto_gerado = None
    if query:
        resultados = [
            p
            for p in POPS
            if query in p["titulo"].lower()
            or query in p["texto"].lower()
            or query in p["modulo"].lower()
            or query in p["subtema"].lower()
        ][:3]
        if resultados:
            pop_destaque = resultados[0]
            texto_gerado = f"Esta é uma resposta gerada pela IA baseada no procedimento '{pop_destaque['titulo']}'. Para realizar esta ação no sistema, recomendo que você siga os passos descritos no vídeo correspondente. O procedimento abrange: {pop_destaque['texto']}"
        else:
            texto_gerado = (
                "Desculpe, não encontrei procedimentos relacionados à sua dúvida."
            )

    return render_template(
        "8_rag_trad.html",
        resultados=resultados,
        texto_gerado=texto_gerado,
        query=request.args.get("q", ""),
    )


# --- APIs para Chat ---
@app.route("/api/search", methods=["POST"])
def api_search():
    data = request.json
    query = data.get("q", "").lower()
    resultados = [
        p
        for p in POPS
        if query in p["titulo"].lower()
        or query in p["texto"].lower()
        or query in p["modulo"].lower()
        or query in p["subtema"].lower()
    ][:3]
    return jsonify({"resultados": resultados})


@app.route("/api/rag", methods=["POST"])
def api_rag():
    data = request.json
    query = data.get("q", "").lower()
    resultados = [
        p
        for p in POPS
        if query in p["titulo"].lower()
        or query in p["texto"].lower()
        or query in p["modulo"].lower()
        or query in p["subtema"].lower()
    ][:3]

    if resultados:
        pop_destaque = resultados[0]
        resposta = f"Esta é uma resposta gerada pela IA baseada no procedimento '{pop_destaque['titulo']}'. Para realizar esta ação, assista ao vídeo no link fornecido."
    else:
        resposta = "Desculpe, não encontrei procedimentos relacionados à sua dúvida."

    return jsonify({"resposta": resposta, "resultados": resultados})


if __name__ == "__main__":
    app.run(debug=True, port=5000)

<script>
    const treeData = {{ tree|tojson }};
    const chatBox = document.getElementById('chatBox');
    
    function addBotMessage(text, options = [], pops = []) {
        const msgDiv = document.createElement('div');
        msgDiv.className = 'chat-msg msg-bot';
        
        let html = `<div class="msg-bubble">${text}</div>`;
        
        if (options.length > 0) {
            html += `<div class="mt-2">`;
            options.forEach(opt => {
                html += `<button class="btn btn-sm btn-outline-primary bg-white me-2 mb-2 fw-semibold" onclick="handleOptionClick('${opt.type}', '${opt.value}', '${opt.modulo}')">${opt.value}</button>`;
            });
            html += `</div>`;
        }
        
        if (pops.length > 0) {
            html += `<div class="mt-2 w-100" style="max-width: 90%;">`;
            pops.forEach(pop => {
                html += `
                <div class="pop-card shadow-sm">
                    <strong class="d-block text-corp">${pop.titulo}</strong>
                    <span class="text-muted d-block mb-2 small">${pop.texto}</span>
                    <div class="row g-2">
                        <div class="col-12 col-xl-6">
                            <a href="#" onclick="fakeDownload(event, '${pop.pdf}')" class="btn btn-sm btn-outline-danger w-100"><i class="bi bi-file-earmark-pdf me-1"></i>Baixar POP</a>
                        </div>
                        <div class="col-12 col-xl-6">
                            <a href="#" onclick="fakeLink(event)" class="btn btn-sm btn-primary w-100"><i class="bi bi-play-circle me-1"></i>Assistir Vídeo</a>
                        </div>
                    </div>
                </div>`;
            });
            html += `</div>`;
        }
        
        msgDiv.innerHTML = html;
        chatBox.appendChild(msgDiv);
        chatBox.scrollTop = chatBox.scrollHeight;
    }
    
    function addUserMessage(text) {
        const msgDiv = document.createElement('div');
        msgDiv.className = 'chat-msg msg-user';
        msgDiv.innerHTML = `<div class="msg-bubble">${text}</div>`;
        chatBox.appendChild(msgDiv);
        chatBox.scrollTop = chatBox.scrollHeight;
    }
    
    function handleOptionClick(type, value, parentModulo = null) {
        addUserMessage(value);
        
        setTimeout(() => {
            if (type === 'modulo') {
                const subtemas = Object.keys(treeData[value]);
                const options = subtemas.map(s => ({type: 'subtema', value: s, modulo: value}));
                addBotMessage(`Certo. Qual sub-tema de ${value} você quer consultar?`, options);
            } else if (type === 'subtema') {
                const pops = treeData[parentModulo][value];
                addBotMessage(`Aqui estão os procedimentos para ${value} em ${parentModulo}:`, [], pops);
                
                // Botão de recomeçar
                setTimeout(() => {
                    addBotMessage("Deseja realizar uma nova consulta?", [{type: 'restart', value: 'Sim, recomeçar'}]);
                }, 1000);
            } else if (type === 'restart') {
                startChat();
            }
        }, 500);
    }
    
    function startChat() {
        chatBox.innerHTML = '';
        const modulos = Object.keys(treeData);
        const options = modulos.map(m => ({type: 'modulo', value: m}));
        addBotMessage("Olá! Por favor, selecione o módulo que deseja consultar:", options);
    }
    
    // Inicia o chat ao carregar
    startChat();
</script>
